# Online Library 

A React-based Online Library application where users can browse books and view details.

## Features
- Book listing with images
- Category-based books
- Browse books by category
- View book details
- Add new books using Redux

## Tech Stack
- React
- JavaScript
- CSS
- Redux
- Git & GitHub

## Project Structure
src

┃ ┗ images
┣ components

┣ redux

##  How to Run the Project

### 1. Clone the repository
```bash
git clone https://github.com/your-username/online-library

```
### 2. Install dependencies
npm install

### 3. Start the app
npm start
